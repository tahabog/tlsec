# 🔒 TLSec – HTTPS Security Auditor

**Find vulnerabilities others miss.**

TLSec is a powerful, open-source HTTPS security scanner that analyzes SSL/TLS certificates, detects weak ciphers, and identifies missing security headers.

## 🚀 Features

- ✅ Certificate analysis (expiry, issuer, SAN)
- ✅ Cipher suite enumeration
- ✅ Vulnerability detection (POODLE, BEAST, FREAK)
- ✅ Security headers check (HSTS, CSP, X-Frame-Options)
- ✅ Professional HTML + JSON reports
- ✅ Multi-threaded scanning
- ✅ Proxy and Tor support
